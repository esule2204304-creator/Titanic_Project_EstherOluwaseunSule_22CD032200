## How to Run

1. Install dependencies:
```
pip install -r requirements.txt
```

2. Train the model:
```
cd model
python train_model.py
cd ..
```

3. Run the app:
```
python app.py
```

4. Open browser: http://127.0.0.1:5000

## Features Used
- Pclass (Passenger Class)
- Sex (Gender)
- Age
- Fare
- Embarked (Port)

## Algorithm
Random Forest Classifier
